Changelog
=========

All notable changes to ``hifigps`` are documented here.

[0.2.0] - 2026-04-24
-------------------

Added
~~~~~
- Documentation with Sphinx (Gallery, Tutorials, API Reference)
- GitHub Actions CI workflow with lint and test
- GitHub Pages deployment workflow for docs
- Gallery page showcasing example figures
- Tutorials page linking to Jupyter notebooks

Changed
~~~~~~~
- CLI migrated from environment variables to argparse

[0.1.0] - 2026-04-24
-------------------

Added
~~~~~
- Initial release
- HI filament stacking simulation utilities
- Galaxy pairwise stacking method
- CLI commands: ``hifigps-convolve``, ``hifigps-stack``, ``hifigps-find-fuzzy``
- Modules: data, plot, calculation, constant, stack, bins, utils, estimate, halo

[0.0.1] - 2025-12-02 to 2026-01-24
-------------------

Added
~~~~~
- Project initialization
- Basic filament stacking framework
- All core features completed (used for arxiv:2411.03988)

**Note:** Versions 0.0.x were used for the paper arxiv:2411.03988. Later versions (0.1.0+) focused on optimization and new project adaptations.
